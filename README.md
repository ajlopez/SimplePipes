# SimplePipes

Simple middleware layer for web applications, inspired by Connect and others. Work in Progress.

## Installation

Via npm using (Node.js)[http://nodejs.org]:
```
npm install simplepipes
```

## Usage

Reference in your program:
```js
var simplepipes = require('simplepipes');
```

## Development

```
git clone git://github.com/ajlopez/SimplePipes.git
cd SimplePipes
npm install
npm test
```
